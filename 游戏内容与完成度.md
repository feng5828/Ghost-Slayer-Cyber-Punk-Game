# 霓虹斩鬼(Ghost Slayer · Cyber Punk)—— 内容系统 / 完成度 / 未来方向

> 版本快照:当前 `main`。技术栈:Three.js(卡通渲染 + 泛光/描边/分调色后处理)+ Rapier3D(物理)+ Vite。纯前端、无后端、无存档。
> 完成度图例:✅ 可玩草案(draft,已成形) · 🟡 部分完成/仅基线 · 🔴 缺失或占位

---

## 0. 一句话概念
赛博中式的黄昏街区里,你是持牌"猎鬼人"。五只常驻鬼**不直接现身**,而是污染周围场景留下线索;你读信号找到它们,**蓄力冲撞打到虚弱 → Q 结界收服**。3 分钟一局,最后 60 秒血月降临、恶鬼反过来猎杀你。核心乐趣来自**涌现式破坏**(火/电/爆炸连锁)与**每只鬼独特的地形解密**。

---

## 1. 世界观与美术设定 —— ✅ draft
- **基调**:新赛博中式 · 黄昏老街。橙红天光 + 冷紫环境光,霓虹招牌/灯笼泛光。
- **渲染**:统一 `MeshToonMaterial` 硬边卡通(三阶色阶)+ 后处理三件套:`UnrealBloom`(霓虹起辉)、`OutlineShader`(Sobel 墨线描边)、`GradeShader`(暗部推冷紫、亮部推暖橙)。地面用 `TOON_GROUND` 锁色相避免被光染色。
- **叙事**:仅靠**菜单说明文字 + 场景氛围**传达,无剧情/过场/世界观文档。🟡 可扩展。
- **文件**:`src/world3d.js`(光影/后处理/地面)、`src/toon.js`(材质与 shader)。

## 2. 场景 / 地图系统 —— ✅ draft(本作最完整的部分)
9×9 格程序化街区(`CELL=22`),中心广场 + 十字工业臂 + 四角街区。
- **五大街区**(各有轮廓剪影、开阔度、地表、常驻鬼):
  | 街区 | 地形 | 常驻鬼 | 剪影 |
  |---|---|---|---|
  | 窄巷老居民区 old | 高墙密巷、筒子楼、晾衣杆 | 蜈蚣精 | 缺角细长 |
  | 庙宇祠堂区 temple | 弧形围墙圆院坝 + 大庙 + 火盆阵 | 鬼火群 | 圆 |
  | 霓虹商业街 market | 成排小商铺/流动摊位/招牌 | 纸傀儡 | 密集方块 |
  | 河道水巷 canal | L 形水网 + 拱桥 + 石堤(下陷感) | 水鬼 | L 形 |
  | 工业机房区 industrial | 电缆 + 电源节点簇 + 格栅/警示地 | 雷鬼 | 十字臂 |
- **每局重拼**:①**换角**——四角街区每局随机排列(朝向无关模块,自动旋转对准中心);②**接缝**——区块之间只保留"几道门",位置每局变;内部布局基本稳定(略抖)。✅
- **行走手感**(仅玩家):各区速度/操控/湿滑不同(庙区最快跟手、老巷黏滞、河道湿滑、涉水拖沓)。✅
- **地表贴片**:工业格栅+警示条、老区磨损、商街霓虹反光、庙区中轴石道、河道湿渍。✅
- **地标**:大庙(朝玩家、弧墙围合)、拱桥(可"走上去"抬升)、中央大灯笼、牌坊山门。✅
- **连通性**:区内优先 DFS 生成树保证全图可达;墙可被撞毁改变导航。✅
- **局限**:有限 9×9 有边界;单张大地面平面(水面无法真正下陷、只能靠石堤造视觉);无地图种子/复现。
- **文件**:`src/village.js`(核心,~1000 行)。

## 3. 玩家 / 猎鬼人 —— ✅ draft(手感为 draft,美术待换绑骨模型)
- **移动**:WASD/方向键;受街区地形手感影响。
- **攻击 = 蓄力冲撞**:按住左键/空格蓄力(越久冲越猛),松开向鼠标方向**突刺**。冲力是唯一伤害来源;冲得越狠拆得越多。近期已把冲刺距离收短、并把建筑/墙做硬(只有重击或连锁能拆楼)。
- **电浆符(E)**:向鼠标掷出,落点起火 + 小范围伤害 —— 纵火把躲藏的鬼逼出来。
- **结界收服(Q)**:置放式场域,锁定附近**虚弱**的鬼为圆心,目标留在场内到进度满即收服;耗**灵力**。
- **灵力经济**:被动微回 + **破坏场景转化**(拆得越狠、灵力越多)—— 破坏是收鬼的弹药。
- **生命/回血**:受击后延迟自动回血;死亡 2 秒后判负。
- **美术**:程序化**人形骨架**(髋/躯干/头/双臂双腿关节)+ 状态步态(待机呼吸、行走摆腿摆臂、蓄力屈膝、冲刺突刺)。玩家反馈"仍略显机械",**计划后期替换为绑骨角色模型**。
- **文件**:`src/creatures/hunter.js`、`src/input.js`。

## 4. 敌人 / 五只常驻鬼 —— 🟡(鬼火已深化,其余为基线 + 地形风味)
**通用 AI(`HiderAI`)**:lurk 潜伏 → flee 逃窜(BFS 走最远格)→ fight 反击(被逼急/血月)。被打到虚弱 8 秒窗口可收服,超时逃逸回满。**地形化 lurk**:各鬼常态贴合主场(见下)。✅

| 鬼 | 主场机制 | AI 地形行为 | 击杀方式 | 视觉 | 完成度 |
|---|---|---|---|---|---|
| **鬼火群 spheres** | 寄生庙区火盆:藏火、缩回续命 | 绕大庙圆周 + 钻进/涌出火盆 | **解密**:先毁所有被占火盆逼出全部火,released 后才能打虚弱→Q 收服 | 青磷火焰簇 + 芯焰火舌 + 火盆火流 | ✅ 深化完成 |
| **蜈蚣精 dragon** | 分节长身、冲刺撞穿墙 | 顺着巷墙切向滑行、沿长巷冲 | 通用:打断体节→虚弱→收服 | 分节 + 背甲 + 复眼/上颚/触须 | 🟡 基线+风味 |
| **纸傀儡 guardian** | 魅化道具/摄魂市民当自爆傀儡 | 商街常态预囤仆从 | 通用:破其纸身→虚弱→收服 | 纸片身 + 墨眼红腮 + 垂纸条手臂 + 头顶符 | 🟡 |
| **水鬼 water** | 水中半透明/免伤/回血 | 巡水网、尽量待水里 | 火/爆炸/冷却泵**逼上岸**再打(已有雏形) | 层叠半透水体 + 水波环 + 水雾 | 🟡 |
| **雷鬼 thunder** | 吸灯柱/电罐的电充能 | 在电源点之间跳站充能 | 玩家**断电**削它,再打虚弱→收服 | 晶体核心 + 尖刺 + 能环 + 电弧 | 🟡 |
- **重生**:任一鬼被收服/消灭后,延迟重生一只**随机种类**(始终维持 5 只)。
- **局限**:除鬼火外,其余四鬼的"解密式击杀"尚未做透(水鬼/雷鬼有机制雏形,蜈蚣/纸傀儡仍是通用打虚弱)。
- **文件**:`src/ai.js`、`src/creatures/{dragon,spheres,guardian,water,thunder}.js`、`base.js`(虚弱/收服/逃逸)。

## 5. 道具与涌现系统 —— ✅ draft(玩法深度的核心)
- **属性驱动**:每个道具只带几个共享属性 —— 可燃 `flammable`、导电 `conductive`、爆炸 `explosive`、带电 `powered`、可魅化 `charmable`、灭火 `douses`、火盆 `brazier`。系统不写死结果,只触发反应。
- **连锁**:破坏带归因 `{owner, chain}`;碎片/被击飞的道具砸坏别的东西 → 连锁深度 +1 → **计分倍率上限 ×4**。这是经济核心。
- **火系统**:点燃 → 持续伤害 → 向邻近可燃物蔓延(连锁+1)、烧生物/市民;火盆常燃、被撞火种四溅;血雨灭火。
- **电火花**:电线塔倒 → 电击生物 + 点燃 + 沿导电物传导。
- **爆炸**:范围伤害 + 冲击波 + 点燃(等离子罐等)。
- **物理**:道具用 Rapier 刚体,可被冲撞击飞、当"炮弹"二次伤害;碎片有寿命。
- **道具清单**:废料捆/全息广告牌/灯柱/等离子罐/水泥墩/盆栽/巨型灯笼/巷墙/居民楼/牌坊/悬浮板车/冷却泵/火盆。
- **文件**:`src/props.js`、`src/damage.js`、`src/fire.js`。

## 13. 未来可推进方向(按优先级)

**A. 最该补的缺口**
1. **音频系统(0→1,收益最大)**:冲撞/收服/起火/爆炸/电击音效,鬼的信号音提示(靠近时的耳语/磷火噼啪),血月/血雨氛围乐,背景音乐。目前**完全没有声音**。
2. **把"解密式击杀"推广到其余四鬼**(承接鬼火的成功范式):
   - 水鬼:必须用火/爆炸/冷却泵把它逼上岸的窗口期击杀(把现有雏形做成明确循环)。
   - 雷鬼:先断掉它正在吸的电源点使其掉电,再收;充满电时几乎无敌。
   - 蜈蚣精:攻击头节/趁冲刺后硬直;或诱它撞穿特定墙自陷。
   - 纸傀儡:先摧毁/夺回它的傀儡,断了"力量来源"才能收。
3. **主角换绑骨角色模型**(用户已确认后期做);顺带给鬼做更好的动画/受击反馈。

**B. 玩法与留存**
4. 新手引导 / 分段教程(信号怎么读、怎么蓄力、怎么收服)。
5. 难度曲线(随时间加压)、局内成长/道具、局外解锁与存档。
6. 平衡性打磨 + 玩测(伤害、收服窗口、灵力经济、连锁收益)。

**C. 场景与内容**
7. 地图**种子/可复现**、更多街区母题、地标互动;(可选)更大或分块流式地图。
8. 天气/时段变体、可破坏性升级、更多涌现反应链。

**D. 工程与平台**

## 14. 如何运行
```bash
npm install
npm run dev      # 本地开发,浏览器打开 http://localhost:5173(需前台聚焦窗口)
npm run build    # 产出 dist/
```
操作:WASD 移动 · 按住左键/空格蓄力松开冲撞 · E 掷电浆符 · Q 展开结界收服。
