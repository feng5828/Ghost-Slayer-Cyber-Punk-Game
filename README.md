# 斩妖村夜 Demon Hunter Village

基于原作《近视测试》(Eye Test Horror) 超自然生物元素重构的 three.js 猎魔游戏。
血月将至的村庄里潜伏着三只中国志怪恶鬼,你是猎魔人——读懂邪气污染的场景信号,
找到它们,在 3 分钟内斩得越多分越高。

## 运行

```bash
npm install
npm run dev     # 开发服务器
npm run build   # 生产构建 → dist/
```

## 玩法

- WASD 移动,鼠标瞄准
- **左键 / 空格**:桃木剑突刺斩(斩妖主力)
- **右键**:掷火符 —— 符落起火,用火焰蔓延把躲藏的恶鬼逼出来、烧穿树篱开路
- 斩杀分递增(400 → 550 → 700…);拆村庄有小分,连锁破坏有倍率;误杀村民 -100

## 三只恶鬼(原作生物的志怪转世)

| 恶鬼 | 原作对应 | 习性 |
|---|---|---|
| **蜈蚣精** | 红龙毛毛虫 | 分节百足,高速掠过撞碎一切;吞噬越多身体越长;逃跑时冲刺留下破坏痕迹 |
| **鬼火群** | 金属球群 | 青磷阴火按 Lissajous 阵型运转;吞噬残骸增殖;被逼急会凝成锥形钻穿树篱;遇电更炽 |
| **纸傀儡** | 友善守护者 | 不直接伤人——怨气附物让器物当傀儡,摄魂令全员扑杀;连村民都能摄魂 |

### 信号系统(捉迷藏核心)

恶鬼不现身,但邪气会污染周围场景:

| 信号 | 含义 |
|---|---|
| 焦热余烬飘起 + 器物低频晃动 | 蜈蚣精在附近潜行 |
| 阴风高频震颤 + 青磷浮动 | 鬼火群蛰伏于此 |
| 杂物被怨气托起漂浮 + 惨白纸灰 | 纸傀儡伪装其间 |
| 屏幕边缘心跳脉动 | 恶鬼已在咫尺 |

### 全局事件

- **血雨**(随机):大地湿滑、火焰熄灭
- **血月**(最后 60 秒):恶鬼停止躲藏、反过来猎杀你,斩杀 ×2

### 涌现规则

道具只挂共享属性(可燃/导电/爆炸/带电/浇灭/可魅化),猎魔人与恶鬼的能力只输出
系统输入(冲量/点火/电击/附身),连锁自己长出来:火符点燃树篱 → 火烧到干草垛 →
草垛滚进火药桶 → 爆炸掀翻电灯柱 → 电火花再点燃一片。水井被炸开会浇灭周围的火。
被斩的恶鬼 4 秒后在远处转世重生(随机种类),猎场永不空缺。

## 结构

```
src/
  main.js            入口/主循环/重生队列/相机
  village.js         程序化 DFS 迷宫村庄 + 圆形碰撞 + BFS 导航
  signals.js         信号系统(粒子/道具微扰/心跳脉动)
  world3d.js         渲染器/灯光/地面(复用原作贴图)
  input.js           键鼠 → 统一 input 结构
  props.js           道具定义/碎片/物理同步
  damage.js          伤害/破坏/连锁归因/爆炸/电火花
  fire.js            火焰蔓延(会灼伤恶鬼)
  score.js           计分:递增斩杀分/连锁倍率/村民惩罚
  events.js          血月/血雨
  critters.js        村民(中立,误杀扣分,会被纸傀儡摄魂)
  ai.js              HiderAI:潜伏/逃窜(BFS走迷宫)/反击
  ui.js              菜单/HUD/结算(DOM)
  creatures/
    base.js          生物基类(移动/眩晕/电击/迷宫碰撞)
    hunter.js        猎魔人(玩家):桃木剑突刺斩/掷火符
    dragon.js        蜈蚣精:分节蛇形跟随(原作算法)
    spheres.js       鬼火群:Lissajous 阵型(原作轨道数学)
    guardian.js      纸傀儡:怨气附物/摄魂令
```
