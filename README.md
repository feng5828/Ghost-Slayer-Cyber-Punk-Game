# 村庄猎杀夜 Grassland Mayhem

基于原作《近视测试》(Eye Test Horror) 超自然生物元素的 three.js 猎杀游戏。
黄昏的程序化迷宫村庄里潜伏着三只超自然生物,你是猎杀者——读懂场景信号,
找到它们,在 3 分钟内杀得越多分越高。

## 运行

```bash
npm install
npm run dev     # 开发服务器
npm run build   # 生产构建 → dist/
```

## 玩法

- WASD 移动,鼠标瞄准
- **左键 / 空格**:突刺斩击(击杀主力)
- **右键**:掷火把 —— 点燃落点,用火焰蔓延把躲藏的生物逼出来、烧穿树篱开路
- 击杀分递增(400 → 550 → 700…);拆村庄有小分,连锁破坏有倍率;误杀村民 -100

### 信号系统(捉迷藏核心)

生物不现身,但会污染周围场景:

| 信号 | 含义 |
|---|---|
| 红色余烬飘起 + 道具低频晃动 | 红龙在附近潜行 |
| 道具高频震颤 + 银色磁尘 | 金属球群蛰伏于此 |
| 杂物诡异漂浮 + 金色光点 | 守护者伪装其间 |
| 屏幕边缘心跳脉动 | 任一猎物已在咫尺 |

### 全局事件

- **血雨**(随机):大地湿滑、火焰熄灭
- **血月**(最后 60 秒):生物停止躲藏、反过来猎杀你,击杀 ×2

### 涌现规则

道具只挂共享属性(可燃/导电/爆炸/带电/浇灭/可魅化),生物与猎人的能力只输出
系统输入(冲量/点火/电击/魅化),连锁自己长出来:火把点燃树篱 → 火烧到干草垛 →
草垛滚进火药桶 → 爆炸掀翻电灯柱 → 电火花再点燃一片。水井被炸开会浇灭周围的火。
被杀的生物 4 秒后在远处重生(随机种类),猎场永不空缺。

## 结构

```
src/
  main.js            入口/主循环/重生队列/相机
  village.js         程序化 DFS 迷宫村庄 + 圆形碰撞 + BFS 导航
  signals.js         信号系统(粒子/道具微扰/心跳脉动)
  world3d.js         渲染器/灯光/地面(复用原作贴图)
  input.js           键鼠 → 统一 input 结构
  props.js           道具定义/碎片/物理同步
  damage.js          伤害/破坏/连锁归因/爆炸/电火花
  fire.js            火焰蔓延(会灼伤生物)
  score.js           计分:递增击杀分/连锁倍率/村民惩罚
  events.js          血月/血雨
  critters.js        村民(中立,误杀扣分,可被守护者魅化)
  ai.js              HiderAI:潜伏/逃窜(BFS走迷宫)/反击
  ui.js              菜单/HUD/结算(DOM)
  creatures/
    base.js          生物基类(移动/眩晕/电击/迷宫碰撞)
    hunter.js        猎杀者(玩家):突刺斩击/掷火把
    dragon.js        红龙:分节蛇形跟随,逃跑时冲刺留痕
    spheres.js       金属球群:Lissajous 阵型,被逼急会钻穿树篱
    guardian.js      守护者:魅化道具/村民,风筝式反击
```
