# 草原大崩坏 Grassland Mayhem

基于原作《近视测试》(Eye Test Horror) 的超自然生物元素重构的 three.js 大乱斗游戏。
红天空降临的三分钟里,操控一只超自然生物,拆光草原上的一切,或毁掉其余生物。

## 运行

```bash
npm install
npm run dev     # 开发服务器
npm run build   # 生产构建 → dist/
```

## 玩法

- **拆迁计时赛**:3 分钟纯刷分。
- **大乱斗**:毁掉其余两只生物获胜(上限 5 分钟)。

通用操作:WASD 移动,鼠标瞄准。

| 生物 | 左键 | 右键 | 成长 |
|---|---|---|---|
| 红龙 | 冲刺撞击(空格亦可) | — | 破坏得多→身体变长,身体即武器 |
| 金属球群 | 按住:钻头突刺 | 按住:环形绞盘 | 吞噬碎片/破坏→球数增加 |
| 守护者 | 按住:凝视魅化道具/孤魂 | 仆从导弹突击 | 仆从即力量,破坏回复核心 |

## 涌现系统

场上道具只挂共享属性(可燃/导电/爆炸/带电/可魅化),生物能力只输出
冲量、点火、电击、魅化这类"系统输入",连锁自己长出来:

- 撞倒电线塔 → 电火花 → 点燃草地 → 火烧到干草垛 → 烧着的草垛滚进油罐 → 爆炸
- 计分规则:直接破坏 ×1,连锁破坏按深度 +0.5 倍率(上限 ×4)——
  计分系统本身教玩家"设计混乱"而不是无脑平A。

全局事件:60 秒后天空变红、结界缩圈;随机血雨(地面湿滑、火焰熄灭)。

## 结构

```
src/
  main.js            入口/主循环/相机
  world3d.js         渲染器/灯光/地面(复用原作贴图)
  input.js           键鼠 → 统一 input 结构
  props.js           道具定义/场地生成/碎片/物理同步
  damage.js          伤害/破坏/连锁归因/爆炸/电火花
  fire.js            火焰蔓延
  score.js           计分与连锁倍率
  events.js          红天空缩圈/血雨
  critters.js        孤魂小人(中立生物)
  ai.js              AI 控制器(与玩家同一 input 接口)
  ui.js              菜单/HUD/结算(DOM)
  creatures/
    base.js          生物基类(移动/眩晕/电击/死亡)
    dragon.js        红龙:分节蛇形跟随(移植原作算法)
    spheres.js       金属球群:Lissajous 阵型(移植原作轨道数学)
    guardian.js      守护者:魅化与仆从导弹
```
